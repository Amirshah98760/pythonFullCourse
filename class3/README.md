# Class 3: Core Data Structures

This folder introduces Python collections: lists, tuples, and dictionaries. The files include examples of creation, access, iteration, and common operations.

## Topics Covered
- Lists: append, insert, remove, pop, sort, reverse
- Tuples: immutability, indexing, count, index
- Dictionaries: key/value access, update, keys/values/items, loops, nested dictionaries

## Files
- list.py: List operations and practice tasks
- tuple.py: Tuple usage and iteration
- dictionary.py: Dictionary operations and practice tasks
