# Class 4: Loops and Sets

This folder covers looping constructs and set operations. It demonstrates for/while loops, control flow in loops, and common set behaviors.

## Topics Covered
- for loops and while loops
- break and continue
- Nested loops and simple patterns
- Sets: uniqueness, add, update, remove, union
- Using sets for duplicates and unique word counts

## Files
- loops.py: Looping examples and patterns
- set.py: Set operations and practice tasks
