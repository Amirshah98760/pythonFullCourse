# Class 1: Basics and Intro Exercises

This folder introduces Python fundamentals through simple scripts and practice tasks. It covers basic printing, variables, data types, arithmetic operations, string handling, input/output, and simple conditional logic.

## Topics Covered
- Printing and multi-line strings
- Variables and data types
- Arithmetic operations
- String length and concatenation
- User input and type conversion
- Basic conditionals (if/elif/else)
- Simple practice problems (swap, averages, areas, parity)

## Files
- first.py: A collection of beginner exercises demonstrating the topics above.
