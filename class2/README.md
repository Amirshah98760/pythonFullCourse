# Class 2: Strings and Conditionals Practice

This folder focuses on working with strings and basic conditional logic. It includes examples of slicing, formatting, common string methods, and simple decision-making problems.

## Topics Covered
- String creation and escape sequences
- Slicing and indexing
- String methods (upper, lower, strip, find, replace, count)
- f-strings and concatenation
- Input handling and validation
- Conditional checks (even/odd, grade, age, password strength)

## Files
- secondClass.py: Practice exercises for strings and if/elif/else logic.
